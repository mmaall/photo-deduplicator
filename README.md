# photo-deduplicator
Pet project to deduplicate my photos. 

## Usage
```bash
 $ ./photo-deduplicator

Usage: photo-deduplicator [-h] [-c value] [-d value] [-L value] [parameters ...]
    -c, --hashingRoutineCount=value
            Number of routines hashing the files. [4]
    -d, --directory=value
            Directory to deduplicate. [photos/]
    -h, --help  
            Help 
    -L, --logFile=value 
            Log file 
```
